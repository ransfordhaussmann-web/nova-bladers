
Import Creator Store / Sketchfab models here for in-game use.

| Folder name | Bey |
|-------------|-----|
| `NovaStriker` | Nova Striker (see docs/SKETCHFAB-NOVA-STRIKER.md) |
| `IronShell` | Iron Shell |
| `CrimsonComet` | Crimson Comet |
| `FrostCrown` | Frost Crown |

After Studio import: ReplicatedStorage → NovaBladers → Models → `<FolderName>`

Optional: set `modelAssets.meshId` in `BeyCatalog.lua` instead of importing a Model folder.
