# Nova Bladers — Bey Model Imports

Import Creator Store or Sketchfab models into Studio under:

`ReplicatedStorage → NovaBladers → Models`

| Model name | Bey | Notes |
|------------|-----|-------|
| NovaStriker | Nova Striker | Sketchfab GLB — see docs/SKETCHFAB-NOVA-STRIKER.md |
| CrimsonComet | Crimson Comet | Creator Store spinning top (optional) |
| FrostCrown | Frost Crown | Creator Store spinning top (optional) |

Alternatively set `modelAssets.meshId` in `BeyCatalog.lua` with a Creator Store rbxassetid.

Procedural fallback models are built automatically when no import is present.
