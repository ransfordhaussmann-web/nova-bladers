## Import

1. Roblox Studio Toolbox → Creator Store → „spinning top“ / „bey“
2. Model unter `Models/` importieren (Name wie oben)
3. Alternativ: `modelAssets.meshId` in `BeyCatalog.lua` eintragen

Ohne importiertes Model wird ein prozedurales Fallback aus `BeyModelBuilder` genutzt.
