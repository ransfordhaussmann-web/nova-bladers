# Bey 3D Models (Studio Import)

Place imported Creator Store or FBX models here. `BeyModelBuilder` clones by `studioModelName` from `BeyCatalog.modelRef`.

| Studio model name | Bey |
|-------------------|-----|
| `NovaStriker` | Nova Striker (Sketchfab GLB — see docs/SKETCHFAB-NOVA-STRIKER.md) |
| `IronShell` | Iron Shell |
| `VoltDash` | Volt Dash |
| `ShadowBite` | Shadow Bite |
| `FrostCrown` | Frost Crown |
| `EmberCore` | Ember Core |

After Studio import: `ReplicatedStorage → NovaBladers → Models → <ModelName>`

Optional: set `modelAssets.meshId` in `BeyCatalog.lua` instead of importing a full model.
