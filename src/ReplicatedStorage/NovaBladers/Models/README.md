# Bey Model Imports

Optional Studio imports under `ReplicatedStorage/NovaBladers/Models/`:

| Folder name | Bey |
|-------------|-----|
| `NovaStriker` | Nova Striker (see docs/SKETCHFAB-NOVA-STRIKER.md) |
| `BlazeComet` | Blaze Comet |
| `CrystalAegis` | Crystal Aegis |
| `TurboNebula` | Turbo Nebula |

**Creator Store:** Toolbox → Creator Store → search `spinning top` → copy MeshId into `BeyCatalog.lua` → `modelAssets.meshId`.

Without import or meshId, procedural 3D layers are built at runtime.
