# Creator Store / Studio Models

Import optional 3D models into this folder for in-game use.
`BeyModelBuilder` checks here first (`modelRef.studioModelName`), then `modelAssets.meshId`, then procedural meshes.

## Expected model names

| Studio model name | Bey |
|-------------------|-----|
| NovaStriker | Nova Striker |
| IronShell | Iron Shell |
| VoltDash | Volt Dash |
| ShadowBite | Shadow Bite |
| CrimsonFang | Crimson Fang |
| FrostCoil | Frost Coil |

After Studio import: `ReplicatedStorage → NovaBladers → Models → <ModelName>`
