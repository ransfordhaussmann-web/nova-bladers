
Optional Creator Store / imported 3D models for in-game Beys.

## Folder names (match `studioModelName` in BeyCatalog)

| Bey | Model folder |
|-----|--------------|
| Nova Striker | `NovaStriker` — see docs/SKETCHFAB-NOVA-STRIKER.md |
| Iron Shell | `IronShell` |
| Volt Dash | `VoltDash` |
| Shadow Bite | `ShadowBite` |
| Crimson Forge | `CrimsonForge` |
| Frost Crown | `FrostCrown` |

After Studio import: `ReplicatedStorage → NovaBladers → Models → <Name>`

Alternatively set `modelAssets.meshId` in `BeyCatalog.lua` (MeshId from Toolbox).
