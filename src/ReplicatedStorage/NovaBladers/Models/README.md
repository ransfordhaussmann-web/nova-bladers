
Creator Store / Sketchfab Bey-Modelle für in-game Nutzung.

## Studio-Import

Nach dem Import in Roblox Studio:
`ReplicatedStorage → NovaBladers → Models → <studioModelName>`

| Bey | studioModelName | Sketchfab (optional) |
|-----|-----------------|----------------------|
| Nova Striker | NovaStriker | siehe docs/SKETCHFAB-NOVA-STRIKER.md |
| Crimson Fang | CrimsonFang | Creator Store → "spinning top" |
| Aurora Mantle | AuroraMantle | Creator Store → "spinning top" |

Alternativ: `modelAssets.meshId` in `BeyCatalog.lua` mit `rbxassetid://…` befüllen.
