
Import optional Creator Store / custom 3D models here for in-game use.

| Model name | Bey | Notes |
|------------|-----|-------|
| **NovaStriker** | Nova Striker | See docs/SKETCHFAB-NOVA-STRIKER.md |
| **BlazeLance** | Blaze Lance | Fire/lance spinning top, ~3.6 studs wide, flat on arena |
| **CoralTide** | Coral Tide | Water/coral spinning top, ~3.5 studs wide |

After Studio import: `ReplicatedStorage → NovaBladers → Models → <ModelName>`

Set **PrimaryPart** (or child `Hull`). The game auto-scales and welds to the physics hull.
