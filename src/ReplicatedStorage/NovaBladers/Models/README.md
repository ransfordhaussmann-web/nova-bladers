Import Creator Store or Sketchfab models here for in-game use.

| Model name     | Bey           | Notes                                      |
|----------------|---------------|--------------------------------------------|
| NovaStriker    | Nova Striker  | Sketchfab GLB — see docs/SKETCHFAB-NOVA-STRIKER.md |
| IronShell      | Iron Shell    | Creator Store → Toolbox → "spinning top"   |
| VoltDash       | Volt Dash     | Optional Creator Store mesh                |
| ShadowBite     | Shadow Bite   | Optional Creator Store mesh                |
| CrimsonBlaze   | Crimson Blaze | Optional Creator Store mesh                |
| FrostOrbit     | Frost Orbit   | Optional Creator Store mesh                |

After Studio import: ReplicatedStorage → NovaBladers → Models → `<ModelName>`

Alternatively set `modelAssets.meshId` in `BeyCatalog.lua` with an rbxassetid.
