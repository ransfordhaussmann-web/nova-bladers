
Import Creator Store / Sketchfab models here for in-game use.

| Model name | Bey | Notes |
|------------|-----|-------|
| `NovaStriker` | Nova Striker | See docs/SKETCHFAB-NOVA-STRIKER.md |
| `IronShell` | Iron Shell | Creator Store spinning top |
| `CrimsonComet` | Crimson Comet | Optional Creator Store mesh |
| `FrostCrown` | Frost Crown | Optional Creator Store mesh |

After Studio import: `ReplicatedStorage → NovaBladers → Models → <ModelName>`

If no model is present, procedural 3D layers are built automatically.
