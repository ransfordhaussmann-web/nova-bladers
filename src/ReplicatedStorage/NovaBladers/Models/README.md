# Bey Models (Studio Import)

Place Creator Store or imported 3D models here for in-game use.

| Model name | Bey |
|------------|-----|
| **NovaStriker** | Nova Striker |
| **CrimsonForge** | Crimson Forge |
| **FrostPrism** | Frost Prism |

After Studio import: `ReplicatedStorage → NovaBladers → Models → <ModelName>`

Procedural fallback models are built automatically when no Studio model is found.
See `docs/BEY-MODELS.md` for Creator Store setup.
