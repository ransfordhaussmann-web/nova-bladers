## Creator Store Beys

Crystal Vortex and Ember Ring support optional Creator Store meshes.
In `BeyCatalog.lua`, set `modelAssets.meshId` after importing from Toolbox.
Search terms are listed in each Bey's `creatorStore.searchTerms`.
