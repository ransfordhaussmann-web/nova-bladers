# Nova Bladers — Bey Models (Creator Store / Sketchfab)

Import spinning-top models into Studio under **ReplicatedStorage → NovaBladers → Models**.

| Studio Name | Bey | Notes |
|-------------|-----|-------|
| NovaStriker | Nova Striker | Sketchfab GLB — see docs/SKETCHFAB-NOVA-STRIKER.md |
| IronShell | Iron Shell | Creator Store import |
| VoltDash | Volt Dash | Creator Store import |
| ShadowBite | Shadow Bite | Creator Store import |
| CrimsonForge | Crimson Forge | Creator Store import |
| FrostPrism | Frost Prism | Creator Store import |

If a model is missing, `BeyModelBuilder` falls back to procedural geometry.
Set `modelAssets.meshId` in `BeyCatalog.lua` to use a Toolbox mesh directly.
