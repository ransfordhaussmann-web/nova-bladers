
Import Creator Store or custom 3D models here for in-game use.

| Model folder | Bey |
|--------------|-----|
| `NovaStriker` | Nova Striker (see docs/SKETCHFAB-NOVA-STRIKER.md) |
| `CrystalBloom` | Crystal Bloom — optional Creator Store mesh |
| `EmberForge` | Ember Forge — optional Creator Store mesh |

After Studio import: ReplicatedStorage → NovaBladers → Models → `<ModelName>`
Set `modelAssets.meshId` in `BeyCatalog.lua` for MeshPart fallback without a Studio model.
