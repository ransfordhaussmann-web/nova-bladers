
Import **NovaStriker** 3D mesh here for in-game use.

## Automatic (recommended)

1. Run `import-nova-striker.bat` on PC → creates `assets/models/NovaStriker.glb`
2. Roblox Studio (Rojo Connected) → **File → Import 3D** → `NovaStriker.glb`
3. **Press Play once** — `NovaStrikerAutoImport.server.lua` moves it here automatically
4. **File → Save to File** — keep the model in your place

## Manual

After step 2, run `tools/nova-striker-import/setup-in-studio.lua` in Command Bar.

## In-game

Pick **Nova Striker** — `BeyModelBuilder` uses this model instead of the procedural fallback.

Sketchfab reference: docs/SKETCHFAB-NOVA-STRIKER.md
