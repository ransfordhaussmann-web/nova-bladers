
Optional Creator Store / imported 3D models for in-game use.

Place each model under `ReplicatedStorage → NovaBladers → Models` with the exact name below.
When present, `BeyModelBuilder` clones the model instead of the procedural fallback.

| Model name | Bey |
|------------|-----|
| `NovaStriker` | Nova Striker |
| `IronShell` | Iron Shell |
| `VoltDash` | Volt Dash |
| `ShadowBite` | Shadow Bite |
| `CrystalVortex` | Crystal Vortex |
| `EmberRing` | Ember Ring |

See `docs/BEY-MODELS.md` and `docs/SKETCHFAB-NOVA-STRIKER.md` for import steps.
