# Bey Models (Creator Store / Studio Import)

Procedural fallbacks exist for all 6 Beys. Optional 3D meshes from Roblox Creator Store:

1. Studio → Toolbox → Creator Store → search "spinning top" / "bey"
2. Import model under `ReplicatedStorage/NovaBladers/Models/` with matching name
3. Or set `modelAssets.meshId` in `BeyCatalog.lua` (rbxassetid)

| Studio model name | Bey | Type |
|-------------------|-----|------|
| NovaStriker | Nova Striker | Attack |
| IronShell | Iron Shell | Defense |
| VoltDash | Volt Dash | Stamina |
| ShadowBite | Shadow Bite | Balance |
| CrimsonBlaze | Crimson Blaze | Attack |
| FrostOrbit | Frost Orbit | Defense |

Sketchfab reference for Nova Striker: see `docs/SKETCHFAB-NOVA-STRIKER.md`
