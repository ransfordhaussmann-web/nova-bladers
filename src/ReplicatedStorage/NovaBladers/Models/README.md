
Import Creator Store / Sketchfab models here for in-game use.

## Studio import

After import in Roblox Studio, place models under:
`ReplicatedStorage → NovaBladers → Models`

| Model name     | Bey           | Creator Store search terms (see BeyCatalog) |
|----------------|---------------|---------------------------------------------|
| NovaStriker    | Nova Striker  | See docs/SKETCHFAB-NOVA-STRIKER.md          |
| CrimsonOrbit   | Crimson Orbit | spinning top, red attack bey, metal top     |
| FrostAnchor    | Frost Anchor  | spinning top, ice defense, blue bey         |

Optional: set `modelAssets.meshId` in BeyCatalog instead of a Studio model folder.
