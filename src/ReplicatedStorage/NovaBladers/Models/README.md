
Optional Studio imports for Creator Store / custom meshes.

Place imported models here (one folder per bey id):

| Model name | Bey |
|------------|-----|
| NovaStriker | Nova Striker |
| IronShell | Iron Shell |
| VoltDash | Volt Dash |
| ShadowBite | Shadow Bite |
| BlazeComet | Blaze Comet |
| FrostPrism | Frost Prism |

After Studio import: `ReplicatedStorage → NovaBladers → Models → <ModelName>`

Alternatively set `modelAssets.meshId` in `BeyCatalog.lua` for a single MeshPart.

See docs/BEY-MODELS.md and docs/SKETCHFAB-NOVA-STRIKER.md
