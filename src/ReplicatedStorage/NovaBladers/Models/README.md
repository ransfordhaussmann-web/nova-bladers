
Creator Store / Studio-Modelle für Nova Bladers.

Importiere Modelle aus dem Roblox Creator Store (Toolbox → Creator Store → „spinning top“)
und lege sie unter `ReplicatedStorage/NovaBladers/Models/` ab:

| studioModelName | Bey |
|-----------------|-----|
| NovaStriker | Nova Striker |
| IronShell | Iron Shell |
| VoltDash | Volt Dash |
| ShadowBite | Shadow Bite |
| CrimsonBlaze | Crimson Blaze |
| FrostOrbit | Frost Orbit |

Alternativ: `modelAssets.meshId` in `BeyCatalog.lua` eintragen (rbxassetid).

Ohne importiertes Modell werden prozedurale 3D-Modelle verwendet.
