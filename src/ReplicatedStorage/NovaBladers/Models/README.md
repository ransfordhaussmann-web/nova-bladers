
Import Creator Store / Sketchfab models here for in-game use.

## Studio model folders

| Folder | Bey | Creator Store search terms |
|--------|-----|---------------------------|
| `NovaStriker` | Nova Striker | spinning top, beyblade attack, pegasus |
| `IronShell` | Iron Shell | spinning top, beyblade defense, metal shield |
| `VoltDash` | Volt Dash | spinning top, beyblade stamina, lightning |
| `ShadowBite` | Shadow Bite | spinning top, beyblade balance, dark |
| `CrimsonOrbit` | Crimson Orbit | spinning top, beyblade attack, fire, solar |
| `FrostAnchor` | Frost Anchor | spinning top, beyblade defense, ice, frost |

After Studio import: `ReplicatedStorage → NovaBladers → Models → <studioModelName>`

Procedural fallback models are built automatically when no Studio model is present.
See `docs/BEY-MODELS.md` for import instructions.
