# Bey 3D Models (Studio import folder)

Place imported Creator Store or FBX models here. Name each model after the bey id:

| Folder name | Bey |
|-------------|-----|
| `NovaStriker` | Nova Striker (see docs/SKETCHFAB-NOVA-STRIKER.md) |
| `IronShell` | Iron Shell |
| `VoltDash` | Volt Dash |
| `ShadowBite` | Shadow Bite |
| `CrimsonForge` | Crimson Forge |
| `AzureTide` | Azure Tide |

After Studio import: `ReplicatedStorage → NovaBladers → Models → <Name>`

Optional: set `modelAssets.creatorStoreId` in `BeyCatalog.lua` to load at runtime without a local copy.
