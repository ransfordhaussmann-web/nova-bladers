# Bey Studio Models

Place imported Creator Store or Sketchfab models here for in-game use.

| Model name | Bey | Notes |
|------------|-----|-------|
| `NovaStriker` | Nova Striker | See `docs/SKETCHFAB-NOVA-STRIKER.md` |
| `IronShell` | Iron Shell | Toolbox → Creator Store → defense/heavy top |
| `VoltDash` | Volt Dash | Flat stamina-style top |
| `ShadowBite` | Shadow Bite | Dark balance-type top |
| `CrystalFang` | Crystal Fang | Ice/attack-style top |
| `BlazeWheel` | Blaze Wheel | Fire/stamina-style top |

After Studio import: `ReplicatedStorage` → `NovaBladers` → `Models` → `<ModelName>`

Procedural fallbacks in `BeyModelBuilder.lua` are used when no model folder exists.
