# Bey Models (Studio Import)

Optional 3D models per bey — import in Roblox Studio under `ReplicatedStorage/NovaBladers/Models/`.

| Folder name | Bey |
|-------------|-----|
| `NovaStriker` | Nova Striker |
| `IronShell` | Iron Shell |
| `VoltDash` | Volt Dash |
| `ShadowBite` | Shadow Bite |
| `BlazeComet` | Blaze Comet |
| `CrystalGuard` | Crystal Guard |

**Creator Store:** Toolbox → Creator Store → search `spinning top` → insert → copy MeshId into `BeyCatalog.modelAssets.meshId`.

**Sketchfab (Nova Striker):** see `docs/SKETCHFAB-NOVA-STRIKER.md`

If no model folder exists, `BeyModelBuilder` builds a procedural layered mesh at runtime.
