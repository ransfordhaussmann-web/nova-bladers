# START HERE — Nova Bladers auf dem PC

**Alles, was wir heute gebaut haben, holst du mit einem Doppelklick.**

---

## Ein Schritt

1. Gehe zu: https://github.com/ransfordhaussmann-web/nova-bladers
2. Grüner Button **Code** → **Download ZIP**
3. ZIP entpacken
4. **Doppelklick auf `START-HERE.bat`**

Fertig. Das Skript:
- lädt Updates von GitHub (falls Git installiert ist)
- prüft dein Setup
- öffnet den Projektordner
- öffnet die **Special-Move-Videos** im Browser

---

## Noch einfacher mit Git (empfohlen)

Falls Git installiert ist (`https://git-scm.com/download/win`):

1. **Doppelklick `START-HERE.bat`** im entpackten Ordner  
   *oder* klont automatisch nach `C:\Users\DEINNAME\NovaBladers`

2. **Cursor** → File → Open Folder → diesen Ordner

3. **Roblox Studio** → Doppelklick `start-rojo.bat` → Plugins → Rojo → Connect

---
