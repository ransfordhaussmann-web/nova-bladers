# Ordner auf dem Laptop nicht gefunden?

Der NovaBladers-Ordner liegt **nicht automatisch** auf deinem PC. Du musst ihn **einmal** holen.

---

## Schnelltest — Windows-Suche

1. `Win + S` drücken
2. Suchen nach: **`NovaBladers`** oder **`nova-bladers`**
3. Nichts? → Ordner existiert noch nicht → unten **Neu anlegen**

---

## Neu anlegen (einfachste Methode — Cursor)

1. **Cursor** öffnen
2. **Strg + Shift + P** → tippe **`Git: Clone`**
3. URL einfügen:
   ```
   https://github.com/ransfordhaussmann-web/nova-bladers
   ```
4. Speichern unter:
   ```
   C:\Users\DEINNAME\NovaBladers
   ```
   (bei dir vermutlich `C:\Users\hp\NovaBladers`)
5. **Open** klicken

Fertig — links siehst du den Ordner mit `src`, `beyblade model`, usw.

---

## Alternative — ZIP (ohne Git)

1. Browser: https://github.com/ransfordhaussmann-web/nova-bladers
2. Grüner Button **Code** → **Download ZIP**
3. ZIP entpacken (Rechtsklick → **Alle extrahieren**)
4. Ordner umbenennen in **`NovaBladers`**
5. Verschieben nach: `C:\Users\hp\NovaBladers`
6. Cursor → **File → Open Folder** → diesen Ordner wählen

---

## Wenn du schon irgendwo Dateien hast

Suche in **Downloads** nach einem Ordner namens:
- `nova-bladers-main`
- `nova-bladers`

Wenn du `import-nova-striker.bat` darin findest → **das ist dein Projekt**.

---
